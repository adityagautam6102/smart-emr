# Clinical UI Design Redesign

## Aesthetic Strategy: "The Formal Clinician"
The UI shift from a generic blue/white theme to a professional **Black-Grey palette** aims to reduce eye strain for clinicians in low-light hospital environments and present a more "serious" medical-grade aesthetic.

### Color Palette
- **Background (Eerie Black)**: `#121212` - Deep, non-distracting foundation.
- **Surface (Raisin Black)**: `#1e1e1e` - Card and modal backgrounds for depth.
- **Primary Accents (Slate/Blue)**: `#3d5afe` - Subdued tech blue for primary buttons.
- **Alert Colors**:
  - Critical: `#cf6679` (Soft Red)
  - Warning: `#ffb74d` (Amber)
  - Normal: `#03dac6` (Teal)

### UI Components
- **Landing Page**: 
  - Glassmorphic hero section.
  - Interactive feature cards.
  - High-contrast typography (Inter/Roboto).
- **Dashboard**:
  - Two-column layout for side-by-side comparison.
  - Sticky header with patient context.
  - Minimalistic charts for lab trends.

### Responsiveness
- Mobile-first approach.
- Collapsible sidebar for tablet/mobile.
- Stacked widgets on small screens.
